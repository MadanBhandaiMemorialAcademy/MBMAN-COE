# Administrator User Guide - MBMAN College Website

Welcome to the administration guide for the Madan Bhandari Memorial Academy Nepal (MBMAN) college website. This guide is designed for non-technical users to help you manage content, update information, and keep the website running smoothly using the **Admin Dashboard**.

## 1. Accessing the Dashboard
*   **URL**: Go to `https://yourwebsite.com/admin/` (e.g., `mbcoe.mbman.edu.np/admin/`).
*   **Login**: Use the username and password provided to you by the technical team.
*   **Dashboard Overview**: Once logged in, you will see cards summarizing the site's content (Total Notices, Faculty, etc.) and quick links to manage different sections.

---

## 2. Managing Notices & Events

### Notices
Notices are announcements displayed on the homepage scrolling marquee and the dedicated Notices page.
1.  **Add a Notice**:
    *   Click **"Add Notice"** on the dashboard or sidebar.
    *   **Title**: The main headline of the notice.
    *   **Date**: Enter the date in Bikram Sambat (e.g., "Kartik 25, 2082").
    *   **Priority**:
        *   *Urgent*: Shows a red warning icon (good for holidays, deadlines).
        *   *Highlight*: Shows in gold/yellow (good for important news).
        *   *Normal*: Standard display.
    *   **File/Image**: Upload a PDF or image if needed (e.g., exam routine).
2.  **Reorder Notices**:
    *   Go to **"Manage Notices"**.
    *   Click and drag the **grip icon** (dots) on the left of any notice row to move it up or down. This changes the order on the public website.

### Events
Events are upcoming activities displayed on the homepage and Events page.
*   **Is Highlight**: Check this box if you want the event to be featured prominently.

---

## 3. Managing Faculty

Faculty members are displayed on the "Faculty" page, organized by department tabs (e.g., "IT Department", "Administration").

### 1. Managing Departments (Tabs)
Before adding faculty, ensure the correct tabs exist.
*   Go to **"Manage Faculty Tabs"**.
*   **Add Tab**: Create a new tab (e.g., Name="Civil Engineering").
*   **Department Filter**: This is a label (e.g., "Civil") used to link faculty to this tab. *Remember this exact text.*

### 2. Adding Faculty Members
*   Go to **"Add Faculty"**.
*   **Department**: Select the department tab from the dropdown list. This determines where they appear on the site.
*   **Social Links**: Add URLs for Facebook, LinkedIn, etc. If left empty, the icon won't show.
*   **Display Order**: You can set a number (1, 2, 3...) to order them manually, or...

### 3. Reordering Faculty
*   Go to **"Manage Faculty Members"**.
*   You will see faculty grouped by their department.
*   **Drag and drop** their cards to rearrange them. The new order saves automatically.

---

## 4. Photo Gallery & Albums

The Gallery allows you to showcase campus life. It is organized into **Albums**.

### 1. Creating an Album
*   Go to **"Manage Albums"** -> **"Create Album"**.
*   Give it a Title (e.g., "Convocation 2025") and Description.

### 2. Uploading Photos
*   Go to **"Manage All Photos"** (or "Add Photo" button).
*   **Image**: Select the photo file from your computer.
*   **Album**: Select the album this photo belongs to.
*   **Caption**: Optional text displayed when hovering over the image.
*   **Use as Album Cover**: Check this if you want this photo to appear on the main album card. You can select multiple covers to create a mini-slideshow for that album.

### 3. Spotlight Images (Homepage)
*   To feature a photo on the **Homepage "In the Spotlight"** section:
    *   When adding/editing a photo, check the **"Feature in Homepage Spotlight"** box.
    *   Or go to **"Manage Spotlight Images"** to see only the featured ones.

---

## 5. Managing Homepage Content

### Hero Slides (Top Carousel)
*   Go to **"Manage Hero Slides"**.
*   This controls the large images and text at the very top of the homepage.
*   You can add multiple slides. They will rotate automatically.
*   **Tip**: Use high-quality images (1920x1080 size recommended).

### Site Configuration (Important!)
This section controls the main identity of your website without needing code changes.
*   Go to **"Edit Page Content"** -> **"Site Configuration"**.
*   **College Name**: Updates the name in the header/footer.
*   **Spotlight Section**: Change the title ("In the Spotlight") and description text shown on the homepage.
*   **Footer Content**: Update the "About Us" blurb and Copyright year in the footer.
*   **Social Media**: Update the links for the social icons in the footer.

### Contact Information
*   Go to **"Edit Page Content"** -> **"Contact Information"**.
*   **Phone/Email**: Updates the clickable contact details across the site.
*   **Google Maps Link URL**: The link opened when clicking "Visit Us" or the address.
*   **Google Maps Embed URL**: The special link for the map display. (See help text under the field for instructions on how to get this from Google Maps).

---

## 6. Programs & Curriculum

*   **Manage Programs**: Add/Edit degree programs (e.g., BIT, Agriculture).
    *   **Brochure**: You can upload a PDF brochure for students to download.
*   **Manage Curriculum**: Add semesters and courses to a program. This appears on the program detail page.

## 7. Frequently Asked Questions

**Q: I uploaded a photo but it's sideways.**
*A: Please edit the photo on your computer to be correct before uploading.*

**Q: How do I delete something?**
*A: In most "Manage" lists, there is a red "Delete" (trash can) icon next to each item. Click it and confirm to remove.*

**Q: The changes aren't showing up!**
*A: Try refreshing your browser (Ctrl+R or Cmd+R). If it still doesn't show, contact support.*

**Q: Can I change the blue color theme?**
*A: The core color theme is part of the site's design code. You can change text content and images, but changing the main colors requires a developer.*
